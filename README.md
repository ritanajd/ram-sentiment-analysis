## Overview

This repository contains the complete, reproducible implementation of the sentiment analysis pipeline developed for the above thesis. The pipeline ingests customer reviews from four platforms, preprocesses and labels them, trains classical machine learning baselines and a multilingual Transformer model (DistilBERT), and produces all figures and evaluation metrics reported in the thesis.

## Results

### Baseline Models (TF-IDF Features)

| Model                  | Accuracy | Macro F1 | Weighted F1 |
|------------------------|----------|----------|-------------|
| Logistic Regression    | 92.21%   | 0.5640   | 0.9033      |
| Linear SVM             | 93.03%   | 0.5484   | 0.9099      |

### DistilBERT Fine-Tuned Model

| Model                              | Accuracy | Macro F1 | Weighted F1 |
|------------------------------------|----------|----------|-------------|
| distilbert-base-multilingual-cased | 90.00%   | 0.5175   | 0.8938      |

**DistilBERT Configuration:**
- Model: `distilbert-base-multilingual-cased` (Sanh et al., 2019)
- Tokeniser: max_length = 64 tokens
- Optimizer: AdamW, learning rate = 2e-5
- Batch size: 8
- Epochs: 3
- Class weights applied to handle severe imbalance
- Hardware: CPU only

> **Note on model choice:** DistilBERT was selected over full mBERT for computational tractability. Sanh et al. (2019) report that DistilBERT retains approximately 97% of BERT's performance on the GLUE benchmark while being 40% smaller and 60% faster. The multilingual tokeniser supports 104 languages including French and Arabic, making the pipeline architecturally ready for future multilingual data.

## How to Reproduce

### 1. Clone the repository

```bash
git clone https://github.com/ritanajd/ram-sentiment-analysis.git
cd ram-sentiment-analysis
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

> **Recommended:** Python 3.9+ in a virtual environment.
> ```bash
> python -m venv venv
> source venv/bin/activate   # On Windows: venv\Scripts\activate
> pip install -r requirements.txt
> ```

### 3. Run the full pipeline

```bash
python run_pipeline.py
```

This will execute all 5 steps in sequence:
1. Data ingestion and harmonisation
2. Sentiment labelling
3. EDA and figure generation
4. Baseline model training and evaluation
5. DistilBERT fine-tuning and evaluation

All outputs are saved to `outputs/`.

### 4. Run individual steps

```bash
python src/01_data_ingestion.py   # Data ingestion only
python src/02_labeling.py         # Labelling only
python src/03_eda.py              # EDA and figures only
python src/04_baselines.py        # Baseline models only
python src/05_distilbert_train.py # DistilBERT training only
```
